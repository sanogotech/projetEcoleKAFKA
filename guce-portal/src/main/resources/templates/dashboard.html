Guide de formation — Bien démarrer sur GUCE CI
Étapes pas à pas
1. Comprendre le contexte

GUCE CI est le Guichet Unique du Commerce Extérieur de Côte d’Ivoire. Il centralise les déclarations douanières (DAU), manifests, paiements, inspections, et échanges avec le port (PCS).

2. Soumettre une DAU

Remplissez l'onglet DAU avec un Correlation ID unique, le déclarant, le bureau de douane, et le payload JSON (ex: marchandises, valeur). Cliquez sur "Soumettre". Un UUID sera généré.

3. Gérer le manifeste

Dans l'onglet Manifeste, saisissez le numéro du manifeste, le navire, la date d'arrivée et les lignes de conteneurs (JSON).

4. Paiement des taxes

Utilisez l'onglet Paiement : calculez les taxes à partir de l'UUID de la DAU, puis confirmez le paiement via un canal (WAVE, MTN, Trésor).

5. Inspection et orientation canal

L'onglet Inspection permet de déterminer le canal (vert, orange, rouge) et de planifier une inspection physique si nécessaire.

6. Suivi portuaire (PCS)

Enregistrez les mouvements de conteneurs (entrée/sortie) et notifiez la mainlevée via l'onglet PCS / Port.
